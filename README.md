**Transform any content into a multi-platform social media empire.** A professional-grade, privacy-first studio that repurposes blog posts, articles, and raw thoughts into optimized, high-engagement content for 6+ platforms simultaneously.

---

## ✨ Premium Features

### 🤖 Autonomous AI Agent Orchestration
This isn't just a wrapper; it's a team of specialized agents working in sequence:
- **🔍 Strategic Analyst**: Extracts hooks, identifies target audiences, and maps out key use cases.
- **⚖️ AI Critic**: Audits the original content for gaps and suggests strategic improvements.
- **✂️ Platform Tailor**: Crafts platform-specific content (Twitter Threads, LinkedIn Articles, Instagram Captions, etc.) while maintaining brand voice.
- **🎨 Visual Designer**: Engineered to generate high-quality visual prompts for AI image generation.
- **📈 Engagement Scorer**: Predicts performance and provides actionable feedback using AI metrics.

### 🖼️ Visual AI Ecosystem
- **Dual Visual Strategy**: Choose between **"Single Unified Image"** for brand consistency or **"Each Platform Unique"** for contextual variety.
- **Native Mockup Previews**: See your content exactly how it will appear on Instagram, Twitter/X, LinkedIn, and YouTube with polished, realistic previews.
- **Gemini Imagen 3 Integration**: Native support for high-quality image generation during the repurposing flow.

### 🔒 Zero-SaaS & Privacy First
- **Local-First Keys**: Your API keys never touch our servers. They are stored locally in your browser's `localStorage`.
- **Bring Your Own Key (BYOK)**: Full control over costs and model selection using **Groq, Gemini, OpenRouter,** or **DeepSeek**.
- **Dynamic Model Management**: The app fetches valid models directly from your provider's API for a seamless setup.

### 🌐 Unified Input & Global Reach
- **OmniInput**: Paste raw text or a **URL** to automatically scrape and analyze web articles.
- **Brand Voice Grounding**: Provide your brand's specific tone, background, and style to ensure every post sounds like you.
- **Multi-Language Support**: Repurpose content from any language to any target language.

## 🚀 Getting Started

```bash
# 1. Clone the repository
git clone <repository-url>

# 2. Install dependencies
npm install

# 3. Start the development studio
npm run dev
```

### Setup Guide
1. Launch the app and head to **Settings**.
2. Select your preferred AI Provider (Groq, Gemini, etc.).
3. Enter your API Key and click **"Verify & Fetch Models"**.
4. Configure your **Brand Voice** and **Target Language**.
5. Enable **Visual AI** to generate professional assets for your posts.
